About the Application

As urbanization increases, crime rates rise, challenging law enforcement to ensure public safety. Traditional reactive policing methods are limited, so this project develops a machine learning-based crime prediction system using historical data. The system uses a Random Forest model with 94.4% accuracy and an RÂ² score of 0.9449, providing reliable crime predictions. It helps law enforcement focus on high-risk areas, improving resource allocation and supporting proactive crime prevention. The Random Forest model outperforms algorithms like SVM and Neural Networks, reducing overfitting and improving predictions. A user-friendly Flask web interface allows input of location, crime type, and time period for real-time predictions. A Twilio-powered alert system notifies authorities when high crime rates are predicted, facilitating timely interventions. This prototype offers a scalable solution for enhancing public safety and resource management.
